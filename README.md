🌍 Wanderlust – Travel Discovery App (MERN Stack)
"Navigate the world with Wanderlust – one destination at a time."

🧭 About Wanderlust
Wanderlust is a full-stack travel exploration web app built using the MERN stack (MongoDB, Express.js, React.js, Node.js). It helps users discover stunning travel destinations, get inspired, and organize their dream getaways—all in one interactive platform.

💡 Why Wanderlust?
Traveling broadens the mind—but planning it can be stressful. Wanderlust brings the beauty of the world to your screen, allowing you to search, save, and explore destinations easily. It's your digital passport to wander freely.

✨ Features
🌍 Discover Destinations – Browse curated places with descriptions and beautiful imagery
🔍 Smart Search – Search by country, place, or category (beaches, mountains, history, etc.)
🧾 Wishlist – Save your favorite spots to revisit later
📱 Fully Responsive – Mobile-first, works on all screen sizes
🔐 Authentication – Secure login/signup (if implemented)
🌐 API Integration – Supports integration with external APIs for data and images
